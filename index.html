من نحن

نحن في بانوراما القصيم نقدم برنامج تدريب تعاوني شامل لطلاب الجامعات في مجالات البرمجة والتصميم.

مراحل التدريب

المرحلة الأولى: التحليل

تتضمن مرحلة التحليل فهم متطلبات المشروع وإنشاء وثيقة SRS (Software Requirements Specification) التي تحدد جميع متطلبات المشروع. يهدف هذا التدريب إلى تعليم الطلاب كيفية تحليل وفهم احتياجات المشروع بشكل دقيق.

المرحلة الثانية: التصميم

في هذه المرحلة، يتعلم الطلاب استخدام أدوات مثل Photoshop وXD لإنشاء نماذج واجهة المستخدم. يساعد ذلك في تخطيط تصميم الموقع وتحديد كيفية تفاعل المستخدمين معه.

المرحلة الثالثة: الفرونت اند

في هذه المرحلة، يتعلم الطلاب HTML وCSS وJavaScript لإنشاء واجهات المستخدم التفاعلية. يتم استخدام HTML لإنشاء هيكل الموقع، وCSS لتنسيق المظهر، وJavaScript لإضافة التفاعلية.

المرحلة الرابعة: الباك اند

في هذه المرحلة، يتعلم الطلاب استخدام إطار عمل Laravel مع لغة PHP لإنشاء الخوادم ومعالجة البيانات. يتضمن ذلك إنشاء قاعدة بيانات وإدارة تفاعلات المستخدم.

المرحلة الخامسة: الاختبار

يتعلم الطلاب في هذه المرحلة كيفية اختبار كل مكونات المشروع لضمان أنها تعمل بشكل صحيح. يتضمن ذلك اختبار الوحدة، اختبار التجميع، واختبار النظام.

الأدوات

Code Editor

يمكنك استخدام أي من المحررات التالية لكتابة كود HTML وCSS وJavaScript الخاص بك:

  • Sublime Text
  • Notepad++
  • VS Code
  • PHPStorm

Local Server

لتجربة تطبيقات الويب الخاصة بك محليًا، يمكنك استخدام أي من الخوادم المحلية التالية:

  • XAMPP
  • MAMP
  • Laravel Herd

GitHub

لتنظيم وإدارة مشاريعك البرمجية، يمكنك استخدام GitHub. يمكن الوصول إلى الموقع من هنا.
